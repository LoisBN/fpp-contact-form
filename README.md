# Contact Form — React Router Actions

Build a contact form that validates input, saves messages to Supabase, and shows success or error feedback.

## Quick Start

**1. Fork this repo** — Click the **Fork** button at the top right of this page.

**2. Clone your fork:**
```bash
git clone https://github.com/YOUR-GITHUB-USERNAME/fpp-contact-form.git
cd fpp-contact-form
npm install
```

Create `.env` with your Supabase credentials, then `npm run dev`.
