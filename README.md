# Contact Form with Actions
A contact form to learn React Router v7 actions for handling form submissions.

## Getting Started
1. Clone: `git clone https://github.com/LoisBN/fpp-contact-form.git`
2. Install: `npm install`
3. Copy `.env.example` to `.env` and add your Supabase credentials
4. Run: `npm run dev`

## Your Tasks
- Create a `messages` table in Supabase
- Implement an action function to handle form submission
- Add form validation (required fields, email format)
- Display success/error feedback after submission
- Show a list of submitted messages
